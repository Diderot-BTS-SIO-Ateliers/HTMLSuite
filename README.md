# Apprentissage de HTML - Suite
Pour créer le dépôt qui contiendra votre travail, cliquez sur ce lien : https://classroom.github.com/a/psZ5Ez8y


Vous devez effectuer 3 exercices, dont les énoncés sont numérotés et donnés dans 3 fichiers pdf.

Pour chaque rendu, faites bien attention aux caractères spéciaux, titre de document, détails de présentation qui font partie de l'exercice.
Certaines images vous sont fournies.

Les fichiers que vous rendez doivent OBLIGATOIREMENT porter votre nom ! Par exemple QuestionnaireNabitz.html
